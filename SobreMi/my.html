Sobre Mí

¡Hola! Soy un desarrollador Fullstack Jr. certificado por Rolling Code School y en Egg. Durante 2 años, me formé intensivamente en tecnologías frontend y backend, aprendiendo a crear aplicaciones web modernas y funcionales. Mi enfoque está en la creación de experiencias de usuario excepcionales y en la resolución de problemas técnicos de manera eficiente.

Habilidades Técnicas

Frontend

  • Maquetación web con HTML5 y CSS3 (Responsive Design con Flexbox y Bootstrap).
  • Desarrollo de interfaces dinámicas con React JS (JSX, Hooks, manejo de estados, ciclo de vida de componentes).
  • Consumo de APIs REST e integración de rutas (rutas protegidas y básicas).

Backend

  • Creación de APIs REST con Node.js y Express.js.
  • Gestión de bases de datos con MongoDB.
  • Autenticación de usuarios con JSON Web Tokens (JWT).

Programación

  • Lógica de programación y estructura de datos (JavaScript: variables, funciones, objetos, DOM, JSON).
  • Manejo de asincronismo (promesas, async/await) y peticiones HTTP.
  • Control de versiones con Git y GitHub.

DevOps

  • Deploy de proyectos frontend y backend en la nube.
  • Administración de repositorios colaborativos con GitHub.

Metodologías y Herramientas

  • Metodología Ágil SCRUM para gestión de proyectos.
  • Colaboración en proyectos grupales e individuales.

Proyectos Realizados

  • Creación de aplicaciones web completas (frontend y backend).
  • Desarrollo de un sistema de carrito de compras y gestión de usuarios con React y Node.js.
  • Diseño y maquetación de páginas web profesionales y responsivas.
