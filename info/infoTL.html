Proyecto Society: Sitio Web para Venta de Cursos

Este sitio web es un prototipo diseñado para mostrar el concepto de una plataforma de aprendizaje en línea.

Los sitios reales serían mucho más complejos, ya que incluirían un backend, frontend, bases de datos y otros sistemas integrados. Este prototipo tiene como objetivo mostrar una visión general, pero los sistemas completos serían mucho más avanzados y robustos.

Si te gustaría ver el sitio web, puedes hacer clic en el botón "Ver Sitio". Para explorar el código en GitHub, haz clic en el botón "GitHub".

Ver Sitio GitHub

Descripción

El proyecto Society incluye funcionalidades avanzadas para mejorar la experiencia del usuario al comprar y gestionar cursos en línea.

Funcionalidades

  • Página de Suscripción: Botón modal que redirige a la página principal (Home).
  • Página Principal (Home):
    • Navbar con menú hamburguesa.
    • Carrito de compras con botón modal para ver los productos seleccionados.
    • Icono de basurero para eliminar compras del carrito.
    • Botón de compra que redirige al menú hamburguesa para ver el resumen de compras.
  • Productos (Cursos): Tarjetas organizadas para mostrar los cursos.
  • Funcionalidad de Admin:
    • Verificación de administrador para eliminar o crear nuevas tarjetas de cursos.

Tecnologías Utilizadas

  • Bootstrap: Para el diseño y la responsividad del sitio.
  • Fuentes de Google: Para la tipografía.
  • Netflix Style: Para el estilo visual de los componentes.
