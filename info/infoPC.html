Sobre el Proyecto

El proyecto se basa en la creación de una página web que fomente el interés por el bienestar y la tecnología avanzada en prótesis.

El diseño y desarrollo incluyen tecnologías como HTML, CSS, Bootstrap, Git, GitHub y metodologías SCRUM.

La meta es construir un sitio innovador que muestre el esfuerzo y dedicación de nuestro equipo.
